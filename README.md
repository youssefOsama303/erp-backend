An integrated, high-performance E-commerce platform coupled with a powerful ERP dashboard for seamless business and inventory management.

## 📖 About The Project

Nexa Store is more than just an e-commerce website; it is a fully functional business ecosystem. It bridges the gap between customer-facing retail and backend operations. When a customer places an order, the inventory updates automatically, sales data is reflected on the admin dashboard, and payments are securely processed.

## 🚀 Key Features

### 🛒 E-Commerce Storefront
* **Dynamic UI:** Smooth Dark/Light mode toggle and full Arabic/English localization (RTL/LTR support).
* **Smart Catalog:** Real-time product filtering by category and brand without page reloads.
* **Seamless Checkout:** Interactive cart system with tax calculation and validation.
* **Payment Integration:** Live integration with **Paymob** for secure credit card transactions and a "Cash on Delivery" option.

### 📊 ERP & Admin Dashboard
* **Data Analysis:** Real-time dashboard for tracking sales, revenue, and inventory levels.
* **Entity Management:** Smart CRM for managing customers and suppliers with automated ID generation and `upsert` database logic to prevent duplicate entries.
* **Inventory Control:** Automated stock deduction and product import scripts.

## 🛠️ Tech Stack

* **Frontend:** Vanilla HTML5, CSS3 (Custom Properties/Variables), JavaScript (ES6+). Built for maximum speed without heavy frameworks.
* **Backend:** Node.js, Express.js.
* **Database:** PostgreSQL (Relational schema with constraint handling).
* **Payment Gateway:** Paymob API.
